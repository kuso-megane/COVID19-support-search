# コロナ支援民間組織の検索、紹介サービス

## 環境
- dockerコンテナを構築し、apache2 + php + mysqlで作成。鋭意進行中！

## frontend
- www/html/views配下にhtml(拡張子は基本php)を記述
- cssは、www/html/asset/stylesheet/配下に
- (scssで書くことも検討中)
- 画像は、www/html/asset/img/配下に

## 外部ライブラリ
- polyfill.js
- [Font Awesome](https://fontawesome.com/)
- あとはwww/composer.jsonみてください
